## 简介

这是一个现代化的 Markdown 到 Word 文档转换器，采用完整的三面板界面系统设计，提供实时预览、配置定制和协作编辑功能。项目基于 TypeScript 构建，使用 Express.js 提供 Web 服务，集成 Collabora 在线办公套件实现文档协作编辑，并通过 LibreOffice 支持 PDF 导出。

**更新** 新增完整的三面板界面系统，包括编辑器列、预览列和配置面板，集成 CodeMirror 编辑器和实时预览功能。预览系统现已扩展到5种模式，包括新增的HTML创意预览模式，支持4种CSS模板风格。

该系统的核心特性包括：
- **三面板界面系统**：编辑器列、预览列、配置面板的完整布局
- **CodeMirror 编辑器**：专业级 Markdown 编辑体验，支持快捷键和语法高亮
- **实时预览功能**：5种预览模式（Markdown即时、HTML创意✨、Docx预览、PDF预览、Collabora编辑）
- **HTML创意预览模式**：4种CSS模板风格（现代暗黑、毛玻璃、复古杂志、赛博朋克）
- **可定制的文档样式配置**：字体、字号、颜色、页面布局等全面配置
- **协作式在线编辑**（Collabora）
- **多格式导出支持**（DOCX、PDF）
- **响应式用户界面设计**：自适应布局和智能面板管理
- **自动预览功能**：防抖处理的智能预览更新

## 性能考虑

### 前端性能优化

   - 使用 CSS Grid 和 Flexbox 实现高效的布局计算
   - 最小化重绘和回流操作
   - 使用 translate 替代 position 属性进行动画

   - 防抖处理减少频繁的预览更新
   - 智能缓存机制避免重复渲染
   - 按需加载编辑器功能模块

3. **HTML创意预览优化**
   - CSS模板预编译和缓存
   - 条件渲染减少DOM操作
   - 模板选择器懒加载

4. **实时预览优化**
   - 多模式预览切换优化
   - PDF 预览的延迟加载
   - Collabora 编辑器的懒加载

5. **自动预览优化**
   - 800ms防抖延迟平衡响应性和性能
   - 编辑器变更和配置变更分别处理
   - 智能预览触发条件判断

6. **资源加载优化**
   - 图片懒加载和尺寸适配
   - CDN 加速静态资源
   - 压缩和合并 CSS/JS 文件

   - 及时清理事件监听器

   - 避免内存泄漏

## 故障排除指南

### 常见问题及解决方案

#### Collabora 在线编辑不可用

**症状**: 预览功能显示 Collabora 不可用

**原因分析**:
- Docker Compose 未正确启动
- 网络连接问题
- 端口冲突

**解决步骤**:
1. 检查 Docker 服务状态
2. 运行 `npm run compose:up` 启动服务
3. 验证端口 3000 是否被占用
4. 检查防火墙设置

#### PDF 导出失败

**症状**: 导出 PDF 时报错 "LibreOffice not found"

**原因分析**:
- LibreOffice 未安装
- soffice 二进制文件路径不正确

**解决步骤**:
1. 安装 LibreOffice
2. 确认 soffice 可执行文件路径
3. 设置正确的环境变量

#### CodeMirror 编辑器加载失败

**症状**: 编辑器无法正常加载

**原因分析**:
- CodeMirror 资源文件加载失败
- JavaScript 错误阻止初始化

**解决步骤**:
1. 检查网络连接和 CDN 访问
2. 查看浏览器控制台错误信息
3. 验证 CDN 资源可用性

#### HTML创意模板加载失败

**症状**: HTML创意预览模式无法显示CSS样式

**原因分析**:
- CSS模板文件加载失败
- 模板名称不匹配
- 样式注入错误

**解决步骤**:
1. 检查网络连接和CDN访问
2. 验证模板名称有效性
3. 查看浏览器控制台CSS注入错误

#### 自动预览功能异常

**症状**: 自动预览不工作或响应迟缓

**原因分析**:
- 防抖函数配置错误
- 事件监听器绑定失败
- 预览模式切换异常

**解决步骤**:
1. 检查防抖延迟设置
2. 验证事件监听器注册
3. 确认预览模式选择器正常

#### 文档生成错误

**症状**: 转换过程中出现异常

**排查步骤**:
1. 检查 Markdown 语法是否正确
2. 验证配置参数的有效性
3. 查看服务器日志获取详细错误信息
